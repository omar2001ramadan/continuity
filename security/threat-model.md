# Threat Model

Status: mainnet-gated checklist.

Tracks protocol, relay, scoring-provider, privacy, graph, Sybil, delegation, and settlement threats. Production launch requires expansion and independent review.

Required evidence:
- Owner: Security lead.
- Status: required before `TSL-MAINNET`.
- Evidence links: STRIDE-style threat inventory, cryptography review, relay/log attack tests, scoring abuse review, and independent audit report.
- Blocking criteria: unresolved critical findings, unaudited signing paths, or mutable closed-log behavior blocks promotion.
