# Key Management Policy

Status: mainnet-gated checklist.

Production relay, provider, auditor, and governance keys require ceremony records, rotation windows, emergency revocation, and recovery tests.

Required evidence:
- Owner: Protocol security lead.
- Status: required before `TSL-MAINNET`.
- Evidence links: key ceremony records, signer inventory, rotation calendar, recovery test transcript, and emergency revocation drill.
- Blocking criteria: dev keys, unowned signers, missing recovery procedure, or untested revocation blocks promotion.
