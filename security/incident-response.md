# Incident Response

Status: mainnet-gated checklist.

Production deployments must define severity classes, paging, containment, disclosure, key revocation, rollback, and post-incident review procedures.

Required evidence:
- Owner: Operations lead.
- Status: required before `TSL-MAINNET`.
- Evidence links: runbook drill records, escalation contacts, incident template, rollback evidence, and post-incident review template.
- Blocking criteria: missing 24/7 contact, missing emergency revocation path, or untested rollback blocks promotion.
