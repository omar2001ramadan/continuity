# Privacy Threat Model

Status: mainnet-gated checklist.

Focus areas: proof bundle disclosure, metadata correlation, graph leakage, provider uploads, appeals, and private evidence retention.

Required evidence:
- Owner: Privacy/security lead.
- Status: required before `TSL-MAINNET`.
- Evidence links: proof-bundle redaction tests, consent vectors, metadata leakage score report, provider upload allowlist review, and retention mapping.
- Blocking criteria: raw content or salts exported without consent, exact counterparty leakage, or missing retention/delete path blocks promotion.
