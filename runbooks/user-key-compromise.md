# User Key Compromise Runbook

Status: draft
Owner: protocol-ops

Detection signals: user report, signed compromise attestation, abnormal revocation request, or provider abuse report.

Immediate response: verify controller or recovery authority, submit revocation, publish resolver state, require step-up for new high-value actions, and preserve evidence commitments.

Recovery evidence required for approval: revocation transaction, resolver propagation proof, affected-key list, user notification record, and post-incident review.
