# Data Retention And Deletion Request Runbook

Status: draft
Owner: privacy

Request classes: local client data, hosted account metadata, proof bundle storage, support records, and abuse evidence.

Immediate response: authenticate requester, classify data placement, delete erasable hosted data, preserve immutable public commitments, and explain limits for third-party copies.

Recovery evidence required for approval: request ticket, data inventory, deletion proof, retained-commitment explanation, and reviewer sign-off.
